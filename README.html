MessageReply Application

MessageReply is an application for posting messages and replying to a particular message.

How to get started:

Load the .sln file in visual studio to run the project.
The UML diagram for this project is given in UML.png
This application provides a simple structure for making a form so that a person can use this project as a backend and develop a GUI on top of it.
